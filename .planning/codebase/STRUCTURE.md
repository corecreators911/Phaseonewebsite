# Project Structure

## Root Directories
- **`src/`**: Main application logic and components.
- **`assets/`**: Static assets (images, fonts, 3D models).
- **`scripts/`**: Utility scripts (e.g., build helpers).
- **`docs/`**: Project documentation.
- **`guidelines/`**: Specific design or development guidelines.
- **`.planning/`**: GSD system files.

## Key Files
- `index.html`: Entry point.
- `vite.config.ts`: Vite configuration.
- `tsconfig.json`: TypeScript configuration.
- `package.json`: Main dependencies and scripts.
