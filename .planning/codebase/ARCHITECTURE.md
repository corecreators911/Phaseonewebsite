# Architecture & Flow

## Overview
The application follows a standard Vite/React architecture tailored for high-performance visual experiences.

## Key Architectural Decisions
- **Vite 6**: Used for fast development and optimized builds.
- **React Router 7**: Handles client-side navigation and page transitions.
- **Tailwind 4**: CSS-first approach for efficient and performant styling.
- **Motion (Framer Motion)**: Primary library for complex UI and element animations.
- **Three.js**: Integrated for specialized 3D scenes (e.g., hero background).
- **Lenis**: Smooth scrolling for polished feel.

## Future Plans
- Component-based design system.
- Shared animation hooks for consistent motion feel.
- Optimized 3D assets for performance.
