# External Integrations

## Libraries & Frameworks
- **Three.js**: Integrated for advanced 3D scenes.
- **GSAP**: Used for high-performance timeline-based animations.
- **Motion (Framer Motion)**: Primary library for complex UI and hover animations.
- **Lenis**: Smooth scrolling for consistent feel across devices.
- **Lucide React**: Icon library for consistent visual language.
