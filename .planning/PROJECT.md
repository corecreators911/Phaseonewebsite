# Project: Phase One VFX Website

## Overview
A premium, immersive website for Phase One VFX studio, featuring advanced animations, 3D elements, and a high-end visual aesthetic.

## Vision
To create a digital experience that reflects the studio's cinematic quality and technical expertise in the VFX industry.

## Core Pillars
1. **Visual Excellence**: Professional-grade design and motion.
2. **Immersive Interaction**: Smooth transitions and 3D integration.
3. **Portfolio Focused**: Clear and compelling showcase of VFX work.

## Primary Stakeholders
- Phase One VFX Team
- Potential Clients (Studios, Directors, Producers)

## Technical Foundation
- **Frontend**: React 18, Vite 6.
- **Styling**: Tailwind CSS 4.0.
- **Animation**: GSAP, Motion (Framer Motion).
- **3D**: Three.js.
- **Routing**: React Router 7.
